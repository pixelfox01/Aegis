# System Requirements

Before installing Raven, ensure your system meets the following requirements.

## Minimum Requirements

### Hardware

- **CPU**: 2 cores
- **RAM**: 2GB
- **Storage**: 10GB available disk space
- **Network**: Internet connection for LLM API access (if using Gemini)

### Software

- **Docker**: Version 20.10 or later
- **Docker Compose**: Version 2.0 or later
- **Operating System**: 
  - Linux (Ubuntu 20.04+, Debian 11+, CentOS 8+)
  - macOS 11.0+
  - Windows 10/11 with WSL2

## Recommended Requirements

### Hardware

- **CPU**: 4 cores or more
- **RAM**: 4GB or more
- **Storage**: 50GB SSD
- **Network**: Stable broadband connection

### For Production

- **CPU**: 8+ cores
- **RAM**: 8GB+
- **Storage**: 100GB+ SSD with backup
- **Network**: Redundant network connections

## Database Requirements

Raven uses PostgreSQL for data storage:

- **PostgreSQL**: Version 16 (included in docker-compose)
- **Storage**: Minimum 5GB for database

## LLM Provider Requirements

Choose one of the following:

### Gemini API (Recommended for beginners)

- Valid Google Cloud account
- Gemini API key
- Internet connection for API calls

### Ollama (For local/offline deployment)

- Additional **8GB RAM** minimum
- Additional **10GB disk space** for models
- GPU recommended but not required

## Network Ports

The following ports must be available:

- **8000**: Backend API (combined deployment) or Backend only (separate deployment)
- **3000**: Frontend (separate deployment only)
- **5432**: PostgreSQL (internal, not exposed by default)

## Browser Support

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## Optional Requirements

### For Development

- Node.js 20+
- Python 3.14+
- Git

<Steps>
  <Step title="Install Docker">
    Visit [docker.com](https://docker.com) and install Docker Desktop for your operating system, or use your package manager:

    ```bash
    # Ubuntu/Debian
    curl -fsSL https://get.docker.com | sh

    # macOS (using Homebrew)
    brew install --cask docker
    ```
  </Step>

  <Step title="Verify Installation">
    Check that Docker is installed correctly:

    ```bash
    docker --version
    docker compose version
    ```
  </Step>
</Steps>

<Warning>
Docker Desktop requires WSL2 on Windows. Make sure to enable virtualization in your BIOS.
</Warning>

### Required Versions

| Software | Minimum Version |
|----------|----------------|
| Docker | 24.0+ |
| Docker Compose | 2.20+ |
| Git | 2.0+ |

## Operating Systems

Raven can run on:

- **Linux** (recommended) - Ubuntu 22.04+, Debian 11+, Fedora 38+
- **macOS** - macOS 12 (Monterey) or later
- **Windows** - Windows 10/11 with WSL2

<Tip>
We recommend running Raven on Linux for production deployments for better performance and resource management.
</Tip>

## Network Requirements

### Ports

The following ports need to be available:

- **3000** - Frontend web application
- **8000** - Backend API server
- **5432** - PostgreSQL database (internal, can be customized)

<Info>
All ports can be customized in your docker-compose.yml configuration.
</Info>

### Firewall

Ensure your firewall allows:
- Incoming connections on ports 3000 and 8000
- Outbound connections for package downloads during setup
